# AGENTS.md

BlogEngine is a simple blog engine. It takes a bunch of markdown files as its input and produces a static website as its output.

## Tech Stack

- C#/.NET 10

## Making changes

- Never modify any files in the `.\.prompts` folder.

## After making changes

- Always run `dotnet format` after making any changes
- Always run `dotnet build` after making any changes
- Always run `dotnet test` after making any changes
- Never commit anything to git
- Do not try to run the application
